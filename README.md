# ZenFlow – Student Meditation App

## Overview

ZenFlow is a student-focused meditation and stress management application designed to help college students maintain mental well-being. This project includes a complete UI/UX redesign, heuristic evaluation, high-fidelity mockups, responsive layouts, and an interactive prototype built using HTML and CSS.

## Features

* Heuristic Evaluation using Nielsen's Usability Principles
* Modern and Student-Friendly UI Design
* High-Fidelity Mobile Screens
* Responsive Desktop Dashboard
* Mood-Based Meditation Sessions
* Stress Tracking and Progress Monitoring
* Achievement Badges and XP System
* Friends Leaderboard and Social Features
* Interactive User Flow Design

## Technologies Used

* HTML5
* CSS3
* UI/UX Design Principles
* Responsive Design
* Prototyping

## Project Screens

* Welcome Screen
* Home Dashboard
* Immediate Calm Session
* User Profile
* Friends & Leaderboard
* Desktop Responsive Layout

## Objectives

The goal of this project is to create a calm, engaging, and accessible meditation experience tailored specifically for students facing academic stress and daily challenges.
