Prototype Navigation Flow · Task 3
FLOW 01
Onboarding Flow
Welcome Screen → Log In / Sign Up → Home Screen. First-time users are taken to sign up. Returning users tap log in and land directly on the home dashboard.
FLOW 02
Immediate Calm Flow
Home → Tap "Immediate Calm" → Session Type Selection → Active Session → Completion Screen → XP reward animation → Return to Home with updated stress %.
FLOW 03
Mood Selection Flow
Home → Tap "What's Your Mood?" → Choose mood (Exam/Sleep/Stress/Love) → Recommended sessions appear → Select duration → Session begins.
FLOW 04
Social & Friends Flow
Home → Bottom nav Friends tab → Leaderboard view → Tap friend profile → View their badges & streak → Send meditation challenge invite.
FLOW 05
Profile & Badges Flow
Home → Bottom nav Profile tab → View XP, streak, and weekly progress → Tap badges section → Full badge gallery → Locked badges show unlock conditions.
FLOW 06
Notification Flow
Push notification received → Tap to open app → Deep-linked directly to session screen → Session completes → Stress level updated → Dashboard reflects change.

Prototype Documentation · Task 3
ANIMATIONS & TRANSITIONS
Screen transitions: Slide left (forward navigation)
Home animation: Breathing circle pulse loop
Button tap: Scale down 0.96 + haptic feedback
XP reward: Confetti burst + counter animation
Badge unlock: Glow reveal animation
Bottom nav switch: Fade + slide icon
USER TESTING FINDINGS
5 college students tested the prototype
4/5 found "Immediate Calm" button instantly
5/5 understood the leaderboard on first view
3/5 wanted a dark/light mode toggle
4/5 said app feels "made for students"
Average task completion rate: 87%
